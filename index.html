License unit visual

Each license unit is either one shared pool Cloud PC or up to three dedicated Cloud PCs.

How the math works

shared units used = shared pool Cloud PCs + Cloud Apps pool Cloud PCs
dedicated units used = ceiling(dedicated Cloud PCs / 3)
total units used = shared units used + dedicated units used
more shared pool PCs = license units left
more dedicated PCs = remaining slots in current dedicated units + 3 per unused unit
This calculator is an explanatory model for Windows 365 Flex capacity planning. Always verify final provisioning behavior in the Microsoft Intune or Windows 365 admin experience.
